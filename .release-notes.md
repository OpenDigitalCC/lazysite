Release 0.5.14 - Central capability resolver (permissions groundwork)

Full suite green; perlcritic clean; strict SBOM gate green.  Channel: edge.

Internal refactor with no behaviour change: every surface that checks permissions
- the manager UI, the control API, the MCP connector, and now the WebDAV endpoint
- resolves through one function (caps_for). Previously WebDAV read per-account
settings on its own path, so group-based grants did not apply to it; now they do.
This is the foundation for explicit channel capabilities (ui / webdav / api / mcp)
and the move of all permissions onto groups.
