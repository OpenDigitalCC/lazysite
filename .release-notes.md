Release 0.5.19 - Analytics over the control API

Full suite green; perlcritic clean; strict SBOM gate green.  Channel: edge.

Visitor analysis (analyse_visitors) is now available over the control API
(?action=analyse_visitors&window=N), gated on the analytics capability - not only
as the MCP tool. So an agent on the API/WebDAV-token channel (e.g. Claude Code)
gets the same sanitised, cached visitor export (aggregates + event stream; no raw
log, filesystem paths, or visitor IPs) that the MCP connector does. Grant the
analytics capability (via a group) to the account that should have it.
