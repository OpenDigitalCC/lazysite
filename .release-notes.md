Release 0.4.77 - Version reflects running code; login niceties

Full suite green (1833); perlcritic clean; strict SBOM gate green.

- [% lazysite_version %] now reports the running processor's OWN version, stamped
  into the .pl at build time, instead of reading .install-state.json (the side-car
  a partial deploy leaves stale). A box running old code now shows its real old
  version on the page - the deploy gap is self-evident. Pairs with install
  --verify (0.4.76).
- Login page: an already-signed-in visitor sees "You are already signed in"
  (Go to site / Sign out) instead of the form; the localhost demo credentials and
  "members area" hint are removed. Cache-safe (client toggle).
