## Fenced-div block Markdown

A heading/list/bold inside a ::: fenced div leaked literal Markdown (## Heading)
because Text::MultiMarkdown treats <div> content as verbatim. convert_fenced_divs
now renders the body (block + inline), skipping a body that nests another fenced
block (include/oembed/form) the later passes must see. A top-level <style> block
is also no longer paragraph-wrapped. Affects every site with boxed headings.

## Auth lifetime in whoami

whoami returns auth: { method: oauth|bearer, expires_at }. For an OAuth session
expires_at is the access-token expiry (~hourly, refreshes transparently) - which
token_expires_at never showed (that is the static-credential expiry, null for a
permanent operator credential). An agent can now see how it is authenticated and
when the session lapses.

## Audit target -> editor

scope than only public pages. Moves link their destination; user-* targets still
open the Users page with that account expanded.
