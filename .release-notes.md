Release 0.4.35 - Dark editor palette; manager-groups picker

Follow-ups from the live review. Full suite green; perlcritic clean; strict SBOM gate
green.

## Readable dark editor (SM116)

The file/content editor (CodeMirror) now has a dark colour scheme tuned for contrast:
headings are a light green (they were a vivid blue, unreadable on the dark surface),
bold is bright, links use the accent, and the harsh red/orange syntax tokens are
softened. Light mode keeps the bundled theme.

## Manager access groups picker (SM114)

The Config page's "Manager access groups" is now a picker - checkboxes of the existing
groups (fetched on load), with a hidden field carrying the comma-separated value the
backend stores. No more typing group names that must match exactly; a note points at
the Users page when no groups exist yet.

## File sort keeps your page

Clicking a column header to sort now re-sorts in place and keeps you on the current
page, instead of resetting to page 1.

## Captured
