Release 0.5.31 - Retire the Config-page "Manager access groups" field

Full suite green (1998 tests); perlcritic clean; strict SBOM gate green.  Channel: edge.

Since the group-capability overhaul, access to the Manager UI is the "Manager UI"
channel capability granted through a group (on the Groups page) - and across sites
only the admins group needs it, which already has it. The old "Manager access
groups" picker on the Config page was a second, redundant way to grant the same
thing, so it is removed.

The underlying manager_groups setting is not gone: it stays a backend-only fallback
in lazysite.conf. Config saves leave it untouched and the engine still honours it -
you just set it in the file (rarely needed) rather than in the UI.
