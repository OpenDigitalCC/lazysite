Release 0.4.79 - Admin bar out of cache; json: UTF-8 fix

Full suite green (1838); perlcritic clean; strict SBOM gate green.

- The manager admin bar is injected per-request at OUTPUT time, never written to
  the shared page cache. It can no longer leak to anonymous visitors, nor vanish
  for a manager who happened to be served from cache (the "bar not always shown"
  bug). Managers use the page cache again (the NOCACHE-for-managers workaround is
  gone).
- json: sources with non-ASCII (em dashes, curly quotes, arrows) used to resolve
  to an empty value - decode_json was being handed an already-decoded string. Now
  the file is read as raw bytes and decoded correctly, so data-driven pages keep
  their typography.
