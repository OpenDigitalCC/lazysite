Release 0.4.70 - Metadata editor + users owner fix

Full suite green (1812); perlcritic clean; strict SBOM gate green.

## Editor

- The page editor's front-matter box is now a first-class "Metadata" editor in its
  own labelled section, stacked above "Content", with a draggable splitter between
  them. A YAML-heavy page (sections:-driven themes) can make Metadata the main
  surface; a prose page shrinks it. Stacked, not side-by-side, since the preview
  is already the side split.

## Users

- "Add user" now offers "Managed by you (operator)" as the default owner, so a new
  account can be owned by the manager at creation - resolving the top-level vs
  managed-by-manager confusion (previously top-level only, then reassign).
