Full suite green (1885); perlcritic clean; strict SBOM gate green.

Form submissions (lazysite/forms/submissions/*.jsonl) were caught by the
lazysite/forms config block - which exists to hide the form CONFIGS (smtp.conf
holds a password) - so the manager file editor opened a submission file EMPTY even
though the records were on disk and delivering ok. The submissions subtree is now
readable (it is append-only data the editor opens read-only); the form configs
stay blocked. The data was never lost - only hidden in the UI.
