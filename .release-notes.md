gate green (182); secrets clean. dist/lazysite-0.4.5.tar.gz built (152 files);
0.4.4 tarball removed.

## Fix: Users/Groups page was scrambled

The SM077 v2 file Access badges (shipped 0.4.2) added a `.mg-acc` CSS rule
(display:inline-block; width:1.1em). That class is also the existing Users/Groups
accordion (`<details class="mg-acc">`), so every user/group row collapsed into a
1.1em inline box and overlapped - "all data on top of itself". The file
access-flag classes are renamed to `mg-rwflag*` (in files.md + manager.css), so
the accordion returns to its normal block layout. No other page was affected.

Operators on 0.4.2-0.4.4 should take this to restore the Users page.
