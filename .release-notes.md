Release 0.4.59 - Layout management over API + MCP

Follow-up to 0.4.58 so the theme agent can manage layouts programmatically. Full
suite green (1751); perlcritic clean; strict SBOM gate green.

## What's new

- Control API (bearer token): layout-install and layout-delete join
  layouts-manifest, gated by manage_layouts.
- MCP connector: three new tools - list_layout_catalogue (browse the repo
  manifest), install_layout (install a layout + its theme(s), activates),
  delete_layout (remove a layout + its themes; refuses the active one).

A partner can now: whoami -> list_layout_catalogue -> install_layout -> activate,
and delete_layout to clean up - the same per-layout flow the Appearance page offers.

Docs: lazysite-layouts/docs/THEME-AGENT.md (theme-author + live-management guide)
and PROCESSOR-INTEGRATION.md (packaging contract).
