Release 0.5.1 - New-site setup fixes

Full suite green (1874); perlcritic clean; strict SBOM gate green.

- Login form: fields stack again (the password field was rendering beside the 2FA
  field due to Markdown <p>-wrapping of a 2-column grid; now a single-column stack).
- setup-manager --link / claim-create print a single-use self-service URL
  (/claim?u=...&c=...) so a new manager sets their own password - nothing to hand over.
- Per-user WebDAV now errors ("WebDAV is not enabled site-wide") unless WebDAV is
  enabled globally, instead of being a dead switch.
- An unauthenticated hit on ?action=logout (e.g. a scanner) no longer writes a
  "logout" audit entry; a real logout now records the actual username.
