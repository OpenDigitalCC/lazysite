Release 0.4.68 - Documentation sweep

Docs-only release: folds the documentation updates into a tag so a deploy carries
them. No code change since 0.4.67. Full suite was green at 0.4.67 (1802);
perlcritic clean; strict SBOM gate green.

## What changed

- Manager docs: "Themes" -> "Appearance" throughout; the Appearance section now
  covers the layout/theme switcher, the repo catalogue, per-layout install/delete,
  and the control-API/MCP equivalents. Config retitled "Site settings". Visitor
  statistics rewritten for the traffic classifier, referrer split, log-path
  privacy + download, and the enable-gated nav item.
- Authoring guides: a "Content components" section (fenced ::: components, slots,
  front-matter sections:).
- Layout-authoring guide (lazysite-layouts): default_theme, the sections variable
  and markdown filter, and a full Content components section.
- Backlog: SM083 v2 note and a 0.4.58-0.4.67 summary.

Also (in 0.4.67) content components became deployable: package-layouts.sh bundles
components/ into the layout zip and layout-install installs the subtree.
