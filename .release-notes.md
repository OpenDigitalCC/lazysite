Release 0.4.67 - Content components deployable via install_layout

Patch so content components actually ship. Full suite green (1802); perlcritic
clean; strict SBOM gate green.

A layout's components/ subtree now travels end to end: package-layouts.sh bundles
it into releases/layouts/<L>.zip (lazysite-layouts), and layout-install installs
it on the site (it previously copied only the root layout files and skipped
subdirectories). A changed component is detected and overwritten with update:true.
