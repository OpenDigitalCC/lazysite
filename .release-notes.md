Release 0.4.73 - json: page-var source

Full suite green (1824); perlcritic clean; strict SBOM gate green.

Build a table/matrix from a JSON file natively. tt_page_var gains a json: source
alongside scan:/url::

    tt_page_var:
      data: json:/data/comparison.json

then loop it in the body: [% FOREACH row IN data.rows %] … [% END %]. Docroot-
contained only; a missing/invalid source resolves to empty so loops degrade
quietly. This is the built-in alternative to [% USE JSON %], which needs the
optional (unbundled) Template::Plugin::JSON. Authoring docs corrected accordingly.
