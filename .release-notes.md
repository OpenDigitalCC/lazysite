perlcritic clean; strict SBOM gate green.

## Finer connector capability by path (SM082)

The MCP file tools (write/replace/move/delete/copy/permissions) are now path-aware,
matching WebDAV's gate: a theme/layout path (lazysite/layouts/**) is authorised by
manage_themes/manage_layouts, while everything else keeps the tool's capability
(manage_content). So a "theme designer" partner can edit theme and layout files through
the connector but is refused content writes (-32002, audited), and a content partner is
unaffected. The requirement is computed from the path at the capability gate.

## Nav URL autocomplete (SM097)

The navigation editor's URL field is backed by a datalist of the site's existing page
URLs - a new manager-api `pages` action walks the docroot for .md/.url pages and maps
each to its clean URL (about.md -> /about, index.md -> /, foo/index.md -> /foo/). It
suggests; it does not restrict, so external links and anchors still work.
