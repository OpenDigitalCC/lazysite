Release 0.4.81 - Security: search-index + visitor-stats hardening

Full suite green (1839); perlcritic clean; strict SBOM gate green.

Two info-disclosure / file-access fixes found in the field:

- /search-index (public JSON, and [% p.path %] in templates) no longer exposes the
  absolute server path of each page (now docroot-relative), and page excerpts strip
  <style>/<script>/comment blocks so their contents don't leak.
- Visitor Stats: the access/error log path is no longer a manager-editable setting
  (a manager could point it at /etc/passwd). It comes from a server-owner env var
  (LAZYSITE_ACCESS_LOG / LAZYSITE_ERROR_LOG, set at install) or auto-detection only.
