Release 0.4.62 - Visitor statistics: traffic classifier + privacy

Reworks the Visitor Statistics plugin. Full suite green (1774); perlcritic clean;
strict SBOM gate green.

## What's new

- Traffic classifier (log-only): every request is classed as a person, the
  logged-in operator, an AI assistant, a bot, or probe noise. The headline
  (visitors, pages, trend, referrers) is the genuine human audience; the other
  classes are reported separately with hit + unique-IP counts. AI/noise/bot
  pattern lists are config-overridable.
- Referrers split into external / internal (self-referrer) / direct.
- Privacy: the access-log disk path is no longer returned to the page; instead an
  operator-only raw-log download is offered (cookie-auth, gated by config).
- Top pages link through to the live page.
- Conditional nav: the "Visitor statistics" item is hidden when the plugin is
  disabled (the processor now exposes enabled plugins to the manager layout).

All classifications are honest log-only estimates, not authenticated identity -
surfaced as such in the UI.
