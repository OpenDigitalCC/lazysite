Release 0.4.84 - Stable default for new sites; SMTP form attachments

Full suite green (1871); perlcritic clean; strict SBOM gate green.

- New sites default to the stable update channel (a fresh install seeds
  update_channel: stable). Existing sites without the key are unchanged ('all') -
  set sensitive ones to stable explicitly.
- The email (SMTP) form handler gains an "Attach uploaded files" option (off by
  default). When on, uploaded files are attached to the notification email and
  listed (name + size) below the message.
