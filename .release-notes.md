Release 0.5.10 - One layout/theme switcher

Full suite green; perlcritic clean; strict SBOM gate green.  Channel: edge.

Switching the active layout or theme now lives in ONE place: Appearance >
"Installed layouts & themes" (click Activate on a layout or a theme; activation
clears the page cache and mirrors theme assets). Removed the two redundant
controls - the Appearance "Active layout & theme" select widget and the Settings
"Active layout"/"Active theme" dropdowns - which used to drift out of sync.
