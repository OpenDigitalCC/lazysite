Release 0.4.80 - Fix manager-page corruption; link-audit restyle

Full suite green; perlcritic clean; strict SBOM gate green.

- CRITICAL fix: 0.4.79's admin bar corrupted the manager UI. Manager pages are
  served behind the auth wrapper (REDIRECT_URL unset), so the bar's manager-page
  skip never fired and it was spliced into a <body> inside the head comment,
  closing the head <script> early. Fixed: resolve the path via REQUEST_URI as
  well, and anchor the bar to the first <body> after </head>.
- The link-audit report (orphaned / broken links) now uses the modern mg-table /
  stat-tile styling instead of plain Markdown tables.
